# Autodoc ENT

Autodoc ENT is a React-based automobile diagnostics and learning platform designed to provide structured technical understanding through interactive modules and assessments.

## Features
- Modular learning (Engine, Transmission, etc.)
- User authentication (localStorage-based)
- Quiz and assessment system
- Interactive UI with dynamic components

## Tech Stack
- React.js
- JavaScript (ES6)
- HTML & CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
